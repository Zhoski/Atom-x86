# 🌌 Atom-x86

**Atom-x86** - это 32-битная операционная система для архитектуры x86, написанная с полного нуля на **C** и **NASM**

## ⚠️ Внимание: Идут работы
> Операционная система сейчас находится на таком этапе, что падает от любого чиха. Больше половины всех моих планов еще не сделаны. Данная версия ОС не тянет даже на демо версию

-----

## 🚀 Roadmap
### Этап 1: Фундамент (Готово ✅)
- [x] Передача управления из загрузчика (Bootloader) в ядро.
- [x] Драйвер VGA текстового режима, настройка GDT и IDT.
- [x] Базовое чтение диска через порты IDE (ATA PIO mode).
- [x] Поиск файлов через статическую таблицу (LBA 90).
- [x] Запуск внешних бинарных файлов (.bin).

### Этап 2: Динамическая ФС (В процессе 🏗️)
- [ ] Реализация записи (`write`) и создания (`create`) файлов.
- [ ] Управление свободным местом на диске (реализация Bitmap).
- [ ] Отказ от жестко заданных LBA-адресов в пользу динамического поиска.
  
### Этап 3: Стабильность и Userland (В планах 📅)
- [ ] Добавление поддержки файлов и создание своего формата исполняемого файла
- [ ] Полноценный интерфейс системных вызовов (`int 0x80`).
- [ ] Пагинация (Paging) и изоляция виртуальной памяти.
- [ ] Многозадачность (вытесняющий планировщик).
- [ ] Продвинутые консольные утилиты (ls, cat, touch, rm).
- [ ] Добавление обработчиков исключений, чистка кода, поддержка разных систем
- [ ] Первая стабильная версия 1.0

## 💻 Как собрать и запустить
*Необходимо: i686-elf-gcc, nasm, qemu.*

1. Скачайте atom_os.img и запустите его командой
   ```bash
   qemu-system-x86_64 -drive format=raw,file=atom_os.img -m 16M
   ```
    Так как ОС нестабильна, я настоятельно рекомендую запускать ОС на 16 мб
